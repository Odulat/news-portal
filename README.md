
## Introducing

News portal Laravel project

## Install

Install dependencies via Composer

``` bash
composer install
```

Migrate tables

``` bash
php artisan migrate
```

Run seeders

``` bash
php artisan db:seed
```

Create symlink

``` bash
php artisan storage:link
```
